# AppCompiler

AI-powered Application Configuration Generator built as part of an AI Engineer Internship Assignment.

## Overview

AppCompiler converts natural language application requirements into structured JSON configurations covering:

- UI Schema
- API Schema
- Database Schema
- Authentication & Roles

The system uses a multi-stage Gemini-powered pipeline consisting of:

1. Intent Extraction
2. System Design
3. Schema Generation
4. Validation & Auto-Repair

## Features

- Multi-stage AI pipeline
- Structured JSON generation
- Cross-schema validation
- Automatic error repair
- Fallback handling for reliability
- Streamlit-based user interface

## Tech Stack

- Python
- Streamlit
- Gemini 2.5 Flash
- JSON Schema

## Internship Assignment

This project was developed as part of an AI Engineer Internship technical assessment to demonstrate:
- Prompt Engineering
- AI Pipeline Design
- System Architecture
- Reliability Engineering
- Full Project Deployment
